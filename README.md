## 自动过视频和阅读任务点（video和document类型）
- 1.配置好python和node解释器
- 2.执行`git clone https://github.com/lzumiKonata-coder/chaoxing_class.git`到本地
- 3.执行`cd chaoxing_class`
- 4.执行`pip install -r requirements.txt`
- 5.执行`npm install crypto-js`
- 6.打开`config.py`配置个人信息
- 7.运行`demo.py`
