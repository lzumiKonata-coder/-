# 自动过视频和阅读任务点（video和document类型）
<h3> 1.配置好python和node解释器
<h3> 2.执行git clone https://github.com/lzumiKonata-coder/chaoxing_class.git到本地
<h3> 3.执行cd chaoxing_class
<h3> 4.执行pip install -r requirements.txt
<h3> 5.执行npm install crypto-js
<h3> 6.打开config.py配置个人信息
<h3> 7.运行demo.py
